## Создание коммита
Для создания коммита используется команда *git commit*. Для этого в терминале с папкой репозитория необходимо написать *git commit -m"<сообщение к комиту>*. Сообщение к коммиту писать **ОБЯЗАТЕЛЬНО!!!**
